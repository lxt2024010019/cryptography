实验报告：流密码多次填充攻击
1. 实验名称
流密码密钥复用攻击（Many-Time Pad Attack）

2. 实验目的
理解流密码加密原理及其核心安全前提：密钥流绝不能重复使用。
掌握利用“空格字符与字母异或”的特性进行密码分析的方法。
通过编程实现自动化攻击，从多条重用密钥的密文中还原目标明文。

3. 实验步骤
数据准备：将提供的 11 段十六进制密文转换为字节数组。
空格探测：两两比对所有密文，检测异或结果是否为合法字母，以此标记潜在的明文空格位置。
密钥恢复：利用确认的空格位置，计算对应位置的密钥流字节。
明文解密：使用恢复的密钥流对目标密文进行异或运算，得到初步明文。
结果修正：结合英语语法规则和上下文逻辑，对初步结果中的个别字符进行校准，确保句子通顺。

4. 实验结果
程序成功破解了目标密文，还原出的完整明文如下：
The secret message is: When using a stream cipher, never use the key more than once
该结果语义通顺，准确揭示了流密码使用的核心安全准则。

5. 实验结论
本实验验证了流密码在密钥复用场景下的严重安全隐患。即使密钥本身具有高强度，一旦重复使用，攻击者无需暴力破解密钥，仅通过简单的统计分析和异或运算即可完全还原明文。这证明了在实际应用中，必须严格保证每次加密使用的密钥流（或 Nonce）是唯一的。
